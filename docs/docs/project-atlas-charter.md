# Project Atlas Charter

## Mission

Atlas is a personal operating system designed to reduce Benjamin's cognitive load by organising work, projects, knowledge and decision making into one trusted system.

---

## Roles

### Chief Engineer (Benjamin)

- Defines objectives.
- Tests functionality.
- Approves or rejects designs.
- Makes high-value decisions.

### Chief Systems Architect (ChatGPT)

- Designs the software architecture.
- Plans development.
- Writes specifications.
- Writes code.
- Recommends technical solutions.
- Carries the implementation burden wherever practical.

---

## Guiding Principles

1. Reduce cognitive load.
2. Simplicity before complexity.
3. Build working software early.
4. Every feature must solve a real problem.
5. Documentation before unnecessary complexity.
6. Benjamin focuses on decisions, not implementation.
7. Atlas is built for Benjamin first.
8. Every development session should leave Atlas measurably better than before.

---

## Development Philosophy

Working software is the priority.

Atlas should evolve through small, reliable improvements rather than large, speculative redesigns.
