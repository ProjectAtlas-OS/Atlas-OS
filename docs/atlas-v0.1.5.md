# Atlas v0.1

## Vision

Atlas is a personal operating system designed to reduce cognitive load by organising work, projects, knowledge and decisions into one trusted system.

Version 0.1 is intentionally simple. Its purpose is to provide a reliable foundation that can be expanded over time.

---

## Objectives

- Always know what to do next.
- Never lose important information.
- Organise work into clear projects and work orders.
- Create a permanent record of completed work.
- Build a system that grows with experience.

---

## Core Features

### Dashboard

Displays:
- Today's Focus
- Active Work Orders
- Waiting On
- Next Action

### Work Orders

Each work order contains:
- ID
- Project
- Title
- Status
- Priority
- Objective
- Checklist
- Notes
- Waiting On
- Parts / Materials
- Photos
- Completion Criteria
- Lessons Learned

### Projects

Projects contain collections of related work orders.

Examples:
- Project Atlas
- Project Bay
- Classic Education
- Mr Spa Tasmania

### Archive

Completed work orders are archived and remain searchable.

### Search

Search all projects, work orders and notes from one place.

---

## Design Principles

- Keep it simple.
- Auto-save everything.
- One source of truth.
- Search before creating duplicates.
- Reduce mental effort.
- Expand only when necessary.
